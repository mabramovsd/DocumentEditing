## 📝 Редактор документов (Pet Project)
Pet-проект для тренировки навыков разработки на ASP.NET Core и JavaScript.
Это Single Page Application (SPA), которое позволяет создавать, редактировать документы и отслеживать историю их изменений.

## 🚀 Возможности
- Аутентификация: Вход в систему с использованием JWT-токенов.
- Управление документами: Создание новых и редактирование существующих документов.
- Контроль доступа (Locking): Система на базе SignalR, которая показывает, кто в данный момент редактирует документ, и блокирует его для других пользователей.
- История изменений (Аудит): Просмотр полной истории редактирования каждого документа.

## 🛠 Технологии
- ASP.NET Core (Бэкенд, API, серверная часть)
- C# (Основной язык программирования)
- JavaScript (нативный,	Логика на фронтенде)
- JWT	Аутентификация и авторизация пользователей
- SignalR	(Реализация real-time функционала (блокировка документа))
- Unit-тесты (Покрытие логики юнит-тестами)

## 🏗 Архитектура
Проект представляет собой SPA-приложение. Изначально был реализован как MVC, но эволюционировал в сторону API + Frontend.

## Контроллеры API:
- [Home] — стартовая страница приложения.
- [Documents] — CRUD-операции с документами.
- [Audit] — получение истории изменений документа.
- [Auth] — аутентификация пользователя.

## 🛠️ Запуск проекта (локально)
- Клонируйте репозиторий:
git clone <ссылка_на_ваш_репозиторий>
- Установите зависимости:
Для бэкенда: dotnet restore
Для фронтенда: npm install
- Настройте адреса директорий в appsettings.json
- Запустите проект:
dotnet run
- Откройте в браузере адрес по умолчанию: https://localhost:5001 (или другой порт).

## 📝 Как использовать
- Вход: Перейдите на страницу логина и войдите в систему под тестовым пользователем (пароля пока нет - вводите что угодно).
- Создание документа: Откройте страницу "документы" и нажмите кнопку "Создать документ".
- Редактирование: Откройте документ. Если его никто не редактирует, вы получите доступ. В этот момент другие пользователи увидят, что документ занят вами.
- История: В интерфейсе документа доступна кнопка "Аудит", которая показывает все предыдущие версии и изменения.
